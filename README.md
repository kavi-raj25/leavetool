🏢 HR Employee Leave Management Tool

A modern and user-friendly **HR Employee Leave Management Tool** built with **React.js** and **Vite**. This application allows employees to apply for leave and enables HR administrators to manage and track leave requests efficiently.

---

📌 Features

- 👤 Employee Leave Application
- 📝 Multiple Leave Types
- 📅 Leave Date Selection
- 📄 Leave Reason Submission
- 📊 Leave Status Tracking
- 🎨 Responsive and Clean User Interface

---

🛠️ Technologies Used

- React.js
- Vite
- JavaScript (ES6)
- HTML5
- CSS3
